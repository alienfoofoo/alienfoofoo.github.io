    和弦进行套路

    ModernMusic
    Created At : 2025-01-27 21:40
    Count:1k Views 👀 :
    Comment:
    1. 1.4536251

    1.4536251

    ​ 作为流行音乐最烂大街的套路，普通的4536251是没什么好说的，这套和弦进行中的任意一个部分都可单独拆出来使用，效果都非常不错。但是普通的进行实在是用的太多太无趣了，这里主要集中于其能产生的变化，并且可以把这些变化的思路平移到别处，从而丰富我们自己的素材库。（以下全以C调为调性）

    1. 四级：
      • 最基本的用法至少也是FM9，so这个音在里面是非常好听的音，增加了和弦的色彩性，尽量把mi和so放在高音部分，更能凸显其色彩。
      • 和弦替换F#m7b5，可以写成Am/F#，后面要接上F7(#11)三全音替换B7
      • 和弦替换Dadd9/#F，后面接Fm6.
    2. 五级：
      • 最基本的用法的ADD9，五级也是能够ADD9的，九音是la这个音在五级和弦也是随便弹，但是七音F是要避免的，由于这里不是属功能，出现了F会让其功能性太强了。
      • 流行歌非常喜欢用一个和弦G/F，这个是一个低音保持，把五级和弦的七音放在最下面，听起来属和弦的色彩并不强烈。
      • 可以使用Dmin/F替换。
      • 使用Fm6和弦替换，这个和弦色彩比较强烈，是一个小和弦，也有把他解释成Ddim7也就是半减七和弦的，因为F6本身就是Dm7的一个转位，Fm6就变成半减七的转位了。替换后会获得一个悲伤的色彩，比较有意思。
      • Fm6可以添加大七音mi构成小大七和弦，也可以添加九音so，取决于旋律在这里是什么，如果加上了最好作为旋律音出现。
    3. 三级：
      • 基本用法就是Em7，小和弦可以无脑弹七和弦，但是三级的九音是#fa，是离调音，fa是降九音，在小和弦的情况下最好不要弹。
      • 一个小替换是Cadd9/E
    4. 六级：
      • 基础款无脑弹Am9，其实六级和弦除了fa以外都可以弹，九音si和七音so可以放在突出的位置。
      • 一个变化就是临时变成大和弦Am -> A，可以是转为A/#C，解决到D或者A衔接后面的二级的根音或七音。相似的还有C#dim7，也就是去掉根音的A7，Am -> G/B -> C -> C#dim7(A7b9) 做一个低音上行。Am -> A -> Edim7 -> bEdim7使用减七和弦加半音下行构造baseline。
      • 替换和弦Cm/bE，使用一个一级小和弦的转位，低音是mi -> bmi的line，总体也是暗淡的色彩，很不错。六级这里旋律经常会唱do，所以可以替换，有种到bE调的离调感。（其实用bE调的六级替换了C调的六级）
      • ！替换和弦bB9(13)，使用b七级的属9和弦替换。比较复杂的用法。本身可以用Eb9(13)，基于一个三全音替换(将二代五，相同的三音七音)的用法替换原来的六级，而bB和E又是三全音的关系。
      • 使用A7alt，也就是A7b9#9#5，去掉五音
    5. 二级：
      • 基础款可以弹Dm9，这个和弦也是几乎可以弹C调的所有音。
      • 升级的可以弹D7 -> D7/#F这样子，变成大和弦或属和弦增加张力。
      • 和弦替换bBadd9，re，fa的旋律因可以替换
    6. 五级：
      • G9加上七音九音，F/G（G9sus4）九挂四和弦都是常见用法。
    7. 一级：
      • 最基本的就是C - C7然后再接上四级，当然了九音随便加，七音酌情加（C7一定要凸显出七音bsi，至于si可以酌情考虑）
      • 可以用F/C -> C 这样的进行结束，古典的套路变格中止。
      • 可以接一个Dm7，C/E在四级前面，构造一个低音上行的线条。
      • 可以替换成Eb69，离调到bE上去。
    内容纯属个人思考与经验，不排除有错误或不足，如果你有你的看法欢迎加我与我进行探讨。
